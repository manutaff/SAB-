== Description ==

*The quickest and easiest way to create testimonial sliders.*

You can quote me on that's intuitive interface allows you to create great looking, mobile-friendly testimonial sliders in minutes with no coding knowledge required. Just fill in the fields, drag and drop them into the order you want and you're good to go.

== Installation ==

Installing "You can quote me on that" can be done either by searching for "You can quote me on that" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I reorder my slides? =

To change the order of the slides simply drag and drop them into place.

= Do I need to style the text on my slides =

The slide text inherits the styling from the theme that is currently active so there is no styling required.
